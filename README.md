## Project Description
This project demonstrates the development of a **simple interactive web dashboard** using the **Dash library in Python**.  
The dashboard visualizes **year-wise sales data** using an interactive **bar chart** created with **Plotly Express**.

Dash allows developers to build web-based analytical applications using only Python, without requiring knowledge of HTML, CSS, or JavaScript.

---

## Objectives
- To understand the basics of the Dash framework
- To visualize data using interactive charts
- To build a simple web-based dashboard using Python
- To demonstrate data-driven decision-making through visualization

---

## Tools & Technologies Used
- **Python**
- **Dash**
- **Plotly Express**
- **Pandas**

---

## Dataset Description
The dataset is created using Pandas and contains the following fields:

| Column | Description |
|------|-------------|
| Year | Year of sales |
| Sales | Sales value for the year |
